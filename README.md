# AI UGC Ad Generator – Salon Matrix

## Overview
This project is part of **Future Interns – Prompt Engineering Task 2 (2026)**.  
The goal is to design a prompt-engineered system that generates high-converting UGC-style ad content for local businesses.

## Business Chosen
**Salon Matrix** – a local salon offering haircuts, styling, hair spa, smoothening, coloring, and makeup services.

## Objective
To create authentic, selfie-style UGC ad scripts optimized for short-form platforms such as:
- Instagram Reels
- Facebook Reels
- YouTube Shorts

## Tools Used
- ChatGPT (Prompt Engineering & Content Generation)

## Folder Structure
- `prompts/` – All prompt templates used
- `outputs/` – Generated hooks, scripts, CTAs, and captions
- `docs/` – Explanation of prompt logic

## Key Highlights
- Problem → Solution → CTA framework
- Multiple prompt variations
- Platform-specific adaptation
- Offer-based urgency scripts

## Outcome
A complete UGC Ad Content Pack demonstrating structured prompt engineering and real-world content marketing use cases.
